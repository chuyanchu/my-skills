# Slide Package Checklist

## Before Generation

- Source files are named clearly.
- Upload instructions explain which files go into NotebookLM.
- Story spine has stable section order.
- Batch plan includes slide numbers and scope.
- Each prompt includes style rules and no-fabrication rules.
- Sensitive source material is removed or replaced.

## During Generation

- Save each batch output separately.
- Keep original prompt with generated output.
- Record failed prompts and regeneration notes.
- Watch for repeated background slides or duplicated concepts.

## After Generation

- Merge slide batches in numeric order.
- Normalize title style, colors, margins, and footers.
- Remove repeated agenda/context slides unless useful.
- Check slide numbers, section transitions, and appendix.
- Verify every strong claim has source support.
- Mark or remove uncertain facts.
- Build a final revision list for manual PPT cleanup.
